Avoid Running out of System Space

Posted on 2024-03-03 • 123 words • 1 minute read
Tags: linux

Check

  • space usage of top pathes
sudo du -sh /* | sort -h
  • how much remain
df -h /
  • if use btrfs
# as superuser
btrfs filesystem usage /
# as normaluser
btrfs filesystem df /

Clear cache regularly

Remove packages that are no longer dependent on them
