waydroid使用技巧汇总

Posted on 2024-06-27 • 28 words • 1 minute read
Tags: linux, wayland

改变手机窗口大小

# 如果是空字符串就是默认大小，可以以数字为参数改变大小
waydroid prop set persist.waydroid.width ""
waydroid prop set persist.waydroid.height ""

app

安装新app

waydroid app install /path/to/your-app.apk

列出已经安装的app

waydroid app list

启动app

waydroid app launch <package-name>
