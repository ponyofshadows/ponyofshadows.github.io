如何启用hugo的数学支持

  1. 在layouts/partials/head.html中添加
  <!-- 引入 MathJax -->
  <script type="text/javascript">
    MathJax = {
      tex: {
        inlineMath: [['$', '$'], ['\\(', '\\)']],
        displayMath: [['$$', '$$']]
      }
    };
  </script>
  <script src="https://cdn.jsdelivr.net/npm/mathjax@3/es5/tex-mml-chtml.js"></script>
  1. 禁用goldmark渲染器 在config.toml中设置
[markup]
  [markup.goldmark.renderer]
    unsafe = true
