Posted on 2024-01-11 • 49 words • 1 minute read
Tags: tools

Example: add a paper to the default library

papis add ~/tmp/Downloads/2103.08123.pdf --from doi https://doi.org/10.1103/PhysRevLett.128.040403 --folder-name quantumn_mechanics/Ruling_Out_Real-Valued_Standard_Formalism_of_Quantum_Theory

If you download PDF papers from journel website such as PRL, you don’t need to specify doi when adding the paper, because it is already included in the paper file.
