Avoid firefox crashing on sway

Posted on 2024-01-10 • 79 words • 1 minute read
Tags: linux, fix
  1. Enter about:config in the address bar
  2. To avoid the crash caused by Incompatibility related to nvidia proprietary graphics driver when opening PDF file or webpages with a quiet amount of resources, set webgl.disabled to True to disable WebGL, a JavaScript API for rendering interactive 2D and 3D graphics on web pages without the need for plug-ins.
  3. Set gfx.webrender.all to True. WebRender is a high-performance rendering engine developed by Mozilla to improve the speed and efficiency of web page rendering.
