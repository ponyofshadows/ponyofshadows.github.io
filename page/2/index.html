文件名讲清所有信息 list下路径命名规范 对于list下任意event, 仿照以下格式记录： 24050809开始记录事件@doc@new|0856 即：（如果是将要发生的事件前面加.）+2位年份+4位日期+2位模糊时间代号+事件标题（根据事件的语言环境选用中英文）+(如果包含一些事件属性，比如有关联的proj, 用@开头；有参考精确时间，用|开头) proj下路径命名规范 对于project, 仿照下面的格式记录： project_name1 .project_name2 ..project_name3 project_name只能用英语，且不能包含这些符号:@;|。 如果一个项目不再会使用，已经进入冷备份阶段，在前面加一个"."， 如果一个冷备份项目要被彻底删除内容，在前面加个".." .bak下路径命名规范 给每个备份盘起名(必须用英文），并在.bak/下建立同名路径作为挂载点。 ls -l即可看出路径的最后改动时间，不需要额外记录。 在路径前加".", 则表示该盘已弃用。 各盘上次备份时间距今超过一周，则可以再备份一次。备份可以基于rsync这个强大的增量备份工具。 从project挑选文件到event的工作流程 event路径下的文件必须按照如下子路径规范存放： ab # 图片和视频 au # 音频 eb # 电子书及相应笔记 pp # 论文及相应笔记 nt # 独立笔记和各种随笔 dt # 实验或调查数据 cd # 代码、配置文件 ot # 其它 event附加的文件遵循的规范 来自project的文件使用硬链接，作为同一个文件，不额外占用空间、不受路径改名的影响。 不在任何event路径下运行任何代码 文件的命名充分表达有用信息。 project的版本控制 使用git进行版本控制。 对于非文本代码项目，可能需要只保存最新的几次提交 可以用变基的方法： git log git rebase -i <第五个提交的哈希值>^ # 启用交互式变基，注意这里的 ^ 是指该提交的父提交。 # 在交互式界面中操作：Git 会展示一个列表，其中包括从指定的提交开始的所有后续提交。在这个界面里，可以选择删除（通过删除行或者使用 drop 命令）早于这五次提交的所有提交记录。 # 解决冲突后完成变基 git rebase --continue # 如果要推送到远程仓库，务必使用--force git push --force origin master 也可以更直接地用重置的方法，同样会改写提交历史：

小学五年级的弟弟的电脑游戏规则
Posted on 2024-02-20 • 75 words • 1 minute read
Tags: diary

基础规则和考试增益 消耗20个游戏币可以在假期玩1分钟电脑游戏。游戏币是实体硬币，其中： 1个绿宝石, 面值：2 1个海盗铜币, 面值：20 1个海盗银币, 面值：100 1个海盗金币, 面值：200 每个星期写完2篇英语阅读后，可以于每天21:40之前在妈妈那里听写单词(10个新学的和随机10个之前背过的)。只要是听写，每写对1个新学单词加20个游戏币，每写错1个任何单词扣除20个游戏币 每学懂1章初中数学，且做对应试卷的成绩有100分及以上（满分120分），可基础获得400个游戏币。如果没有100分以上，但有90分以上，可获得200个游戏币。在某一章没有获得过100分以上时可以申请重做一张新试卷，这时候如果获得100分以上奖励200个游戏币。某一章获得过一次100分以上，或者获得过2次90分以上后，就可以开始做下一章的试卷。 任何有完整题目的语文考试中，如果成绩93分及以上（满分100分），可固定获得500个游戏币奖励。 期中考试和期末考试中，若语文93分以上，科学97分以上，道法97分以上，分别获得持续一个月的“思想起源”(学初中数学、背初中单词、抽背语文小升初试卷变动游戏币数量加算20%)、“牛顿的祝福”(学初中数学赚取游戏币数量加算10%)、“莎士比亚的祝福”(背初中单词变动游戏币数量加算10%) 期中考试、期末考试数学或英语若获得满分，都可以获得一个抽卡券，通过抽卡能直接获得大量游戏币或获得大幅增益 获得500游戏币 （5%概率） 获得1000游戏币 （20%概率） 获得1500游戏币 （5%概率） 获得2000游戏币 （5%概率） 接下来一周学习初中数学和听写英语词汇游戏币变动数量加算 50% （25%概率） 接下来一周学习初中数学游戏币赚取数量加算 120% （20%概率） 接下来一周听写英语词汇游戏币变动数量加算 120% （20%概率） 生日当天系统赠送一个金币（200游戏币） 每周可以领取2次小升初语文试卷并记诵答案，然后上交试卷，24小时后，可以在妈妈那里抽查上交的语文试卷。从去除了作文的70分题目中抽取30分的题目进行提问。每答对1分获得15个游戏币。如果妈妈发现一周都没抽查语文试卷，可要求扣除一个银币（100游戏币） 语文摘抄本每增加一页，在妈妈签字后可以领取两个铜币 每次因为和游戏币相关的缘由动怒，就要扣除一个银币。 诅咒系统 开局自带永久诅咒“上帝之骰”，每周都无法自己选择玩的游戏，只能在我的世界、原神、求生之路2、艾尔登法环中随机抽取 开局自带永久诅咒“沉重枷锁”，每周最多只能玩40分钟电子游戏。 如果有超过1周没有听写单词，会获得诅咒“一忘皆空”，必须先听写30个记过的单词，才能开始在听写新单词中获得游戏币收益。 等级系统 初中数学有6个学期，初中英语词汇也分为6个部分。 初始等级是0。当初中数学学完某个学期，且初中英语单词也从开头背到相应的部分，就能升到对应的等级。比如数学学完七年级下册，词汇背完前两部分，就可以升到第2级。 升级时可以花费游戏币在技能树中点一个技能点。以下是不同级别的技能点花费的游戏币数量: 1级 1000个游戏币 2级 1000个游戏币 3级 2000个游戏币 4级 2000个游戏币 5级 4000个游戏币 6级 4000个游戏币 技能树 点完一个线路前面的技能才能点后面的技能： 自由市场→银行贷款→实业投资 纵情声色→卧薪尝胆→劳逸结合 [线路一] 自由市场: 移除“上帝之骰”诅咒，可以自由选择想玩的游戏。 银行贷款: 贷款2600游戏币，接下来两周(从周一开始)每天还款200游戏币。每个月最多只能发动一次。 实业投资: 花费800游戏币，接下来一周(从周一开始) 学初中数学、背单词、抽背语文试卷变动游戏币数量增加60%。每个月最多只能发动一次。 [线路二] 纵情声色: 移除“沉重枷锁”诅咒，假期最长游戏时间不再限制。 卧薪尝胆: 发动后下周(从周一开始)学习初中数学、背单词、抽背语文试卷游戏币变动数量增加40%，但是下周任何时候都不能玩电脑游戏，也无法使用手机娱乐。每个月最多只能发动一次 劳逸结合: 即使不是假期，也能消耗游戏币玩电脑游戏，但非假期每天游玩时间上限是40分钟。
