第一次使用github pages

新的开始

今天是我第一次使用github pages部署我的博客！学习任务重，我必须一天三次小总结、一次大总结，以保证学习质量。

hugo的日常使用

  1. 在archetypes路径下创建文章的模板
  2. 使用hugo new <type name>/<article name>命令新建文章
  3. 在content目录下编辑文章。如果完稿，把"draft"属性设置为false。
  4. 发布文章前，可以运行hugo server预览文章效果，该命令运行成功后即可在http://localhost:1313/ 中预览网站.
  5. 执行hugo生成HTML网页，保存网页文件的路径是public\。
  6. 通过git把public\下的文件push到<username>.github.io仓库

启用额外功能

  • 参考github-style的配置模板，增加了一些功能。
  • 也参考了front-matter的官方文档。

一些笔记：

  1. 在front-matter区域写katex = "math"或mathjax = true以启用KaTex或MathJax的数学渲染，前者速度更快，后者对latex语法支持更广泛。需要注意，行内公式的语法类似\\C \\)。
